# Axiomatic BI

I build analytics systems that teams can own after handover.

My focus is analytics engineering: ingestion, transformation, semantic modelling, and reporting delivery built around clear ownership and reproducible workflows.

## What I Offer

- **Data pipeline implementation** using contract-first ingestion and transformation patterns.
- **Semantic and dimensional modelling** for consistent, governed metrics.
- **Reporting delivery** in Power BI and programmatic outputs using shared model contracts.
- **Technical handover** with documentation, reproducibility steps, and explicit trade-offs.

## How I Work

- Version-controlled delivery from source to report.
- Business logic lives in the model, not hidden in dashboards.
- Evidence-backed claims, supported by code paths, tests, and artefacts.
- Honest scope: demonstration vs production-ready is always clearly labelled.
